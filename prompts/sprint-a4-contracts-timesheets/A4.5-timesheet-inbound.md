## Modèle
```typescript
interface Timesheet {
  id, agencyId, externalTimesheetId,
  workerId, missionContractId, clientId,
  workDate, plannedStart, plannedEnd,
  actualStart, actualEnd, breakMinutes,
  surchargesApplied: Surcharge[],
  totalMinutes, totalCostRappen,
  anomalies: Anomaly[],
  state: 'received' | 'under_review' | 'signed' | 'disputed' | 'tacit',
  receivedAt, reviewedAt
}
```

## Tests
- Event reçu → timesheet créé, anomalies détectées.
- Pause 0 min sur 8h travaillées → anomalie `missingBreak`.
- 51h cumulées sur la semaine → anomalie `weeklyLimitExceeded`.
- Validation tacite : état passe à `tacit`.
