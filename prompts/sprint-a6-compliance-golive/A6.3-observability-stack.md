## Configuration importante
- Sentry région **EU** (DPA signé).
- Pseudonymisation dans les logs : pas de `firstName+lastName` en clair.
- Sampling traces : 10% prod, 100% dev.

## Alertes P1 initiales
- API down > 5 min → on-call
- Taux 5xx > 2% sur 5 min → on-call
- Queue BullMQ backlog > 1000 → on-call
- Batch paie hebdo failed → on-call immédiat

## Tests
- Erreur simulée → Sentry capture avec bon context.
- Trace OTel d'un appel API → visible dans Grafana Tempo.
- Alerte simulée → pager reçu.
