## Widgets
1. **LSE** : autorisation cantonale (active/expire dans X j), autorisation fédérale si applicable.
2. **CCT** : barèmes à jour ? date dernière mise à jour, changements attendus.
3. **Documents workers** : % workers avec tous docs valides, liste des expirations imminentes.
4. **Missions actives** : count, détail par client, dépassement LTr éventuels.
5. **nLPD** : registre à jour, DPIA présente, dernière audit, droits des personnes pending.

## Tests
- Autorisation LSE expirée → widget rouge + CTA "renouveler".
- Barèmes CCT datant > 13 mois → widget orange "à vérifier".
- Worker avec permis B J-60 → apparaît dans liste expirations.
