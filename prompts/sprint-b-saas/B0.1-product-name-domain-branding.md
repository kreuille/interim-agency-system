## Contexte

Le produit a été baptisé "Helvètia Intérim" lors de la PR #71 qui a posé le design system. On formalise ce choix comme nom commercial éditeur SaaS et on réutilise le sous-domaine `helvetia-interim.guedou.ch` du fondateur (décision actée par ADR-0006 §3, zéro coût, zéro délai).

## Instructions

1. **Pas d'acquisition de TLD** : la décision est de rester sur `guedou.ch` pour simplicité. Migration vers TLD dédié = dette future, pas ce sprint.

2. **DNS** (à configurer via le gestionnaire DNS actuel de `guedou.ch`, probablement Infomaniak ou équivalent) :
   - `helvetia-interim.guedou.ch` → CNAME ou A racine vers landing (Cloudflare Pages ou GCP Cloud Storage statique).
   - `www.helvetia-interim.guedou.ch` → CNAME → landing.
   - `app.helvetia-interim.guedou.ch` → A → GCP Cloud Run web-admin (staging puis prod).
   - `api.helvetia-interim.guedou.ch` → A → GCP Cloud Run API.
   - `m.helvetia-interim.guedou.ch` → A → GCP Cloud Run web-portal intérimaire PWA.
   - `docs.helvetia-interim.guedou.ch` → CNAME → Mintlify/GitBook (pour B4.2).
   - `status.helvetia-interim.guedou.ch` → CNAME → Instatus (pour A6.7 et B4).
   - Certificats Let's Encrypt via GCP managed certs ou Cloudflare — tous ces sous-domaines sont des niveaux 3 qui passent sans souci avec Let's Encrypt (limite à 63 chars respectée).
   - SPF/DKIM/DMARC à configurer sur `helvetia-interim.guedou.ch` indépendamment de `guedou.ch` personnel pour ne pas mélanger la réputation email perso et SaaS.

4. **Package branding** :
   ```
   packages/branding/
     src/
       logo.svg                  # full color
       logo-mono.svg             # monochrome pour backgrounds colorés
       icon.svg                  # icône seule
       colors.ts                 # export { helvetiaRed, slate, ... }
       typography.ts             # export font stack
     tailwind.config.ts          # extend { colors, fontFamily }
     index.ts
   ```
   Consommé par `apps/web-admin`, `apps/web-portal`, et future `apps/landing`.

5. **Palette recommandée** (à valider par fondateur / designer) :
   - Primaire : `#D8232A` (rouge suisse, rappel drapeau) OU `#003DA5` (bleu confédération) — trancher.
   - Secondaire : `#1E293B` (slate-800) — texte principal.
   - Accent : `#F59E0B` (amber-500) — CTA secondaires, badges.
   - Succès : `#10B981` — vert émeraude.
   - Erreur : `#DC2626`.
   - Neutre : `#F8FAFC` (slate-50) fond, `#E2E8F0` (slate-200) borders.

6. **Typographie** :
   - Titres : **Inter Display** ou **IBM Plex Sans** (serveur via @fontsource, pas Google Fonts en direct — nLPD).
   - Corps : **Inter** ou **IBM Plex Sans**.
   - Mono : **JetBrains Mono** ou **IBM Plex Mono**.

7. **Tagline / positionnement** :
   - Commerciale courte (10 mots max) : "Le SI suisse natif pour agences d'intérim".
   - Sous-titre élévateur (25 mots) : "Gestion intérimaires, disponibilités temps réel, paie hebdomadaire CCT, facturation QR-bill, intégré par API avec MovePlanner. Hébergé en Suisse. Conformité LSE incluse."
   - Une version plus "growth" à tester : "L'outil que vos intérimaires n'ont plus besoin de vous appeler pour utiliser."

8. **Kit presse minimal** dans `docs/branding/`:
   - `logo-full-color.svg` + PNG 512px + 1024px.
   - `logo-monochrome.svg` + PNG.
   - `icon.svg` + PNG 512px (pour favicon, app icons).
   - `palette.md` avec hex codes.
   - `description-court-fr.md` (50 mots).
   - `description-long-fr.md` (150 mots).
   - `founder-bio-fr.md` (biographie fondateur 80 mots, style LinkedIn).
   - `screenshot-dashboard.png` (une capture web-admin propre, données fictives).

9. **Update assets** :
   - `apps/web-admin/public/favicon.*` à partir du nouveau kit.
   - `apps/web-portal/public/icon-{192,512}.svg` mis à jour.
   - `apps/web-portal/public/manifest.webmanifest` : name + short_name + theme_color.
   - `README.md` racine : ajouter bandeau "Helvètia Intérim — SaaS post-pilote".
