## Contexte

Stripe est le standard SaaS international, support CH mature, Stripe Tax automatique pour la TVA 8.1%, facturation sortante QR-bill compatible via export. Alternatives CH (Datatrans, Wallee) acceptées mais moins de DX pour un monorepo Node.

## Instructions résumées

1. Ouvrir compte Stripe.com → mode CH → fournir IDE + IBAN + identité fondateur (KYC).
2. Activer **Stripe Tax** pour la CH (TVA 8.1% auto sur les factures).
3. Créer Produits dans Stripe dashboard :
   - `helvetia-starter` : 199 CHF/month, trial 14 jours.
   - `helvetia-pro` : 499 CHF/month, trial 14 jours.
   - `helvetia-enterprise` : priced at runtime via API (custom).
   - Versions yearly avec -15% : `helvetia-starter-yearly` (2'030 CHF/an), `helvetia-pro-yearly` (5'100 CHF/an).
   - Add-ons : `sms-pack-1000` (25 CHF ponctuel), `signature-qualified` (par unit).
4. Configurer Customer Portal Stripe (permet aux clients de changer plan, voir factures, update CB).
5. Webhook endpoint : `api.helvetia-interim.guedou.ch/webhooks/stripe` avec HMAC Stripe, events souscrits :
   - `customer.subscription.created`
   - `customer.subscription.updated`
   - `customer.subscription.deleted`
   - `invoice.paid`
   - `invoice.payment_failed`
   - `checkout.session.completed`
6. Secret webhook → `stripe-webhook-secret` en Secret Manager GCP.
7. Pour MVP : endpoint stub qui log et retourne 200. Impl réelle du dispatcher en B2.1.
