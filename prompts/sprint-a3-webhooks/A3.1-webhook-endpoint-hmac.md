# A3.1 — Endpoint /webhooks/moveplanner avec HMAC + tolérance ±5min

> **DoD** :
> - [ ] Route `POST /webhooks/moveplanner` en `express.raw({ type: 'application/json', limit: '256kb' })`
> - [ ] Vérif HMAC-SHA256 constant-time, sur `timestamp.rawBody`
> - [ ] Timestamp ±5 min tolérance, au-delà 401 + log sécu
> - [ ] 2 secrets acceptés simultanément (rotation 90j)
> - [ ] Endpoint `/webhooks/moveplanner/_health` retourne les versions de secret acceptées
> - [ ] Rate limit par IP MP (allowlist IP + 1000 req/min)

## Tests
- Signature valide → 200.
- Signature invalide → 401, log sécu `webhook.hmac.invalid`.
- Timestamp +6 min → 401.
- Body JSON légèrement modifié après signature → 401 (constant-time).
- 2 secrets valides : signature avec v1 OK, v2 OK, v3 pas encore actif → 401.
