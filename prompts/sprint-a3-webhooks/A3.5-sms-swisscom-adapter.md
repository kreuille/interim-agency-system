## Tests
- Envoi OTP → provider noop → log en DB.
- STOP reçu via webhook → flag opt-out sur worker, plus d'envois.
- Rate limit atteint → err RateLimited.
- Template avec variable manquante → err TemplateError, pas d'envoi.
