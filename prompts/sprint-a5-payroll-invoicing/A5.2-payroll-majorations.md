## Table
```
canton_holidays
  canton char(2)
  holiday_date date
  label text
  paid bool default true
  PRIMARY KEY (canton, holiday_date)
```

## Tests
- 1er août → férié fédéral tous cantons.
- Jeûne fédéral → 3e lundi septembre, payé dans majorité cantons.
- Genève → Escalade 12/12 spécifique.
- Vaud → lundi de Pâques férié payé.
- Combinaison nuit+dim+férié → majorations cumulées par branche.
