## Règles d'UI
- **Date format** `fr-CH` `dd.MM.yyyy`.
- **Montants** via `Intl.NumberFormat('fr-CH', { style: 'currency', currency: 'CHF' })`.
- **Erreurs** : toast + message accessible sous le champ.
- **Loading** : skeleton shadcn, pas de spinner plein écran.

## Tests
- Playwright : login → navigation → création worker avec AVS → voir dans liste.
- Rôle viewer : bouton "Nouveau worker" absent.
- Formulaire AVS invalide → message d'erreur accessible.
- Upload doc > 10 MB → message explicite.
