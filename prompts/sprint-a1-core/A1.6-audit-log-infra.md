## Schéma DB
```
audit_logs
  id uuid pk
  agency_id uuid not null
  actor_user_id uuid null        -- null si système
  actor_kind enum(user, system, webhook_mp, job)
  entity_type text not null
  entity_id uuid not null
  action enum(created, updated, archived, deleted, signed, validated)
  diff jsonb not null             -- { before, after } partial
  occurred_at timestamptz not null default now()
  prev_hash text not null
  entry_hash text not null
  ip inet null
  @@index([entity_type, entity_id])
  @@index([agency_id, occurred_at desc])
```

Triggers Postgres : `REVOKE UPDATE, DELETE ON audit_logs FROM app_user`.
