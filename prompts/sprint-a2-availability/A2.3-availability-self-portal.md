## Flows clés
1. Premier accès : saisir email → magic link → session 30j (refresh).
2. Voir 2 semaines, taper sur jour → bascule statut (disponible / indisponible).
3. Synchro auto toutes les 5 min si onglet actif.
4. Offline : bouton "mode offline", modifs queue localStorage, banner "2 modifs en attente".

## Sécurité
- Magic link single-use, expire 15 min.
- Pas de données sensibles en cache offline (juste slots + nom worker).
- CSP stricte.

## Tests
- Lighthouse PWA score ≥ 90.
- Offline basique : désactiver réseau → toujours interactif → reconnexion → sync.
- Accessibilité Axe : 0 violation critical.
