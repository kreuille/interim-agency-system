## Librairies
- `react-big-calendar` (défaut) ou `fullcalendar/react` — choisir en ADR rapide. Préférer react-big-calendar (plus léger).

## Interactions
- Drag sur zone vide → modale création slot (horaires, statut, raison).
- Clic sur slot existant → modale édition.
- Shift-clic → sélection multiple → action groupée.

## Tests Playwright
- Créer slot 8h-17h → apparaît en vert.
- Créer indispo sur même plage → remplace, orange/rouge selon type.
- Import CSV (2 colonnes date/statut) → slots créés.
- Resp mobile : calendrier scrollable horizontalement.
